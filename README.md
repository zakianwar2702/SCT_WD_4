This project is developed as part of SkillCraft Technology Web Development Internship Task 4.
A basic To-Do web application built using Django that allows users to:
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Set due date and time
- Login authentication

Technologies Used
- Python
- Django
- HTML
- CSS
- GitHub

How to Run
1. Clone the repository
2. Open terminal in project folder
3. Run:
python manage.py runserver
4. Open browser:
http://127.0.0.1:8000/
